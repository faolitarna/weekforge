<progression-protocol>

Operationalized progression rules. Signal-based decisions are in feedback-interpretation; safety overrides are in guardrails. This file covers step sizes, ceilings, and graduation.

<volume-load-ladder>

**Rep steps:** +2 reps per step (e.g., 8 → 10 → 12)
**Set steps:** +1 set per step (e.g., 3x → 4x)
**Volume ceiling:** 4 sets x 12 reps (exercise-specific overrides below)
**Load increase:** +2kg upper body, +2.5–5kg lower body — then reset to 3x8

**Progression sequence (lower body):**
3x8@20kg → 3x10@20kg → 3x12@20kg → 4x8@20kg → 4x10@20kg → 4x12@20kg → 3x8@25kg

**Progression sequence (upper body):**
3x8@12kg → 3x10@12kg → 3x12@12kg → 4x8@12kg → 4x10@12kg → 4x12@12kg → 3x8@14kg

**Timed exercises:** +5–10sec per step, set steps same as above
**Bodyweight:** Add reps/sets until ceiling → add external load (vest, belt, dumbbell)

**Volume ceiling overrides:**

| Exercise type | Ceiling | Rationale |
|---------------|---------|-----------|
| Heavy compounds (deadlift, squat) | 4x8 | Load increase at lower volume — joint and recovery cost is high |
| Isolation / accessory | 4x15 | Higher rep ceiling before load increase |
| Isometrics | 4x45sec | Time-based ceiling |

</volume-load-ladder>

<exercise-graduation>

When an exercise hits its equipment or safety ceiling, graduate to the next variation.

| Current exercise | Graduates to | Trigger |
|------------------|-------------|---------|
| Goblet squat | Front squat | DB exceeds ~28–30kg or becomes awkward to hold |
| KB deadlift | Trap bar deadlift | KB weight insufficient for progression |
| DB press | Barbell press or heavier DB | Assess spine safety before graduating |
| Bodyweight pull-ups | Weighted pull-ups | 4x10 BW reached |
| BW dips | Weighted dips | 4x12 BW reached (if shoulder-safe) |
| Band face pulls | Cable face pulls → increase weight | Band resistance maxed |
| Single-leg exercises | Increase ROM before load | e.g., deficit step-ups before adding weight |

</exercise-graduation>

<focus-exercise-progression>

**Loaded focus (standard ladder applies):**
- Bicep curls, elevator press, single arm OHP, carries, pull-ups, face pulls

**Bodyweight / timed focus (progress duration → add load):**
- Bar hangs: +5sec per step → add weight belt
- Side planks: +10sec per step → add hip dip reps → add weight

**Movement quality focus (progress complexity → add load):**
- Reverse/multidirectional lunges: master pattern → add load → add deficit
- X-Press Lat Walk: increase band resistance → add distance

</focus-exercise-progression>

</progression-protocol>
